# activcorner

## Setup 
```bash
> pipenv install django~=3.1.0
> pipenv shell
```

## Create SuperUser
```bash
> python manage.py createsuperuser
```

## Run project
```bash
> python manage.py runserver
```
Access it on http://127.0.0.1:8000


Create and modify course(s) and registration(s) on http://127.0.0.1:8000/admin/
